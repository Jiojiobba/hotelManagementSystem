# hotelManagementSystem

#### 项目介绍
小型宾馆管理系统

#### 软件架构
该项目主要使用技术为: jQuery + layui


#### 安装教程
1.jQuery 版本选用 1.9
2.layui 版本选用2.4.5



#### 使用说明

前端界面模板

1.命名统一：

首字母小写的驼峰型，id 用“_”分割，class 用“-”分割，如：id="userName_main" class="userName-main"
js 文件统一放置在 js 文件夹，css 统一放置在 css 文件夹，资源文件统一放置在 resource 文件夹


2.Git操作须知：

每个人新建自己的分支
将自己负责的模块推到自己的分支后，再推到主分支融合
不要轻易改动主分支
